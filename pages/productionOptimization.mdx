# Optimizing for Production

If you want to bring your project to the production version, you should definitely check out the following documentation and best practices:

- [Building for Production](https://vitejs.dev/guide/build.html) (Vite)
- [Optimizing for Production](https://tailwindcss.com/docs/optimizing-for-production) (Tailwindcss)
- [Optimizing Performance](https://reactjs.org/docs/optimizing-performance.html) (React)

<br />

> Since Overline does not require any special configurations, you do not need
> to pay extra attention to the template settings, but can optimize the project using
> the documentation mentioned above.
